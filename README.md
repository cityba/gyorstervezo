# gyorstervező app
Gyorstervező, alap funkciókkal, főfal, válaszfal, ablak, ajtó. floorplanner import fml fájl készítéssel, így be lehet importálni egy alap rajzot , amit 3D ben látható ezután a floorplanner oldalon.

Quick planner, with basic functions, main wall, partition wall, window, door. Floorplanner import fml file creation, so you can import a basic drawing, which can then be seen in 3D on the floorplanner page.
